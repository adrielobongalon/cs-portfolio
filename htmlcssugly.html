HTML

HTML is like a skeleton. It forms the basic structure of the webpage and tells the computer where content goes, like which paragraph goes where and and in what order to put them. HTML stands for HyperText Markup Language, and it was invented in 1980 by a physicist named Tim Berners-Lee. HTML is used by all websites, including this one. It supports the embedding of text, images, and other content commonly found on websites. The most recent version of HTML (as of writing) is HTML5, a revision of previous versions. It was released in October of 2014. Usually, the web browser doesn't display the actual HTML document, but rather, it displays the content that the document tells it to show.

The language of HTML consists of text and/or information surrounded by HTML tags (or just "tags", for short), which look like letters or words enclosed between angled brackets. The picture to the upper left shows an example of a tag, the <html> tag. This tag tells the web browser, "The HTML document begins here." </html> is what the corresponding closing tag would look like. Notice how the only difference between the opening and closing tags is the forward slash that precedes the tag.

Here's a list of some common HTML tags.

tag stands for usage
<html> hypertext markup language indicates the start and end of the HTML document
<body> body body of the page, most of the webpage content goes here
<p> paragraph used for most text and paragraphs
<h1> heading, size 1 used for headings, size can be changed to whole numbers 1-6
<img> image used to insert images
<table> table used to create tables
<div> & <span> division & span used to group content, div and span have slightly different uses that
